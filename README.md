# Python Projects Collection 🐍

## Overview
This repository contains a collection of **small Python projects** designed to demonstrate skills in data analysis, visualization, file handling, and application development.  
Each project is self‑contained, easy to run, and highlights practical problem‑solving with Python.

---

## Projects Included
- **Student Performance Tracker** → Manage grades, hobbies, and profiles with file handling.  
- **Smart Contact Directory** → CRUD operations for storing and searching contacts.  
- **Inventory Management System** → Track items, billing, and stock using lists + files.  
- **Healthcare Data Cleaner** → Clean messy patient datasets with Pandas.  
- **Bike Sharing Analytics** → Analyze raw station usage data with NumPy/Pandas.  
- **Course Popularity Explorer** → Udemy dataset analysis with Python + Power BI dashboards.  
- **Titanic Survival Insights** → Data analysis and visualization using NumPy, Pandas, Matplotlib.  

---

## Features
- Modular Python scripts for different scenarios  
- Data cleaning and preprocessing with Pandas  
- Visualization with Matplotlib, Seaborn, Plotly  
- File handling and persistent storage examples  
- Recruiter‑ready dashboards and reports  

---

## Outcomes
- **Practical Applications**: Each project solves a real‑world scenario (education, healthcare, inventory, analytics).  
- **Data Insights**: Cleaned datasets, visualizations, and KPIs for decision‑making.  
- **Portfolio Value**: Demonstrates versatility across domains and tools.  
- **Recruiter‑Friendly**: Clear structure, professional naming, and concise documentation.
